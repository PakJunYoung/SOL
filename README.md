# SOL
> Survey Of Love(설문조사 프로그램)_2인개발

netbeans jFrame을 이용하여 GUI를 만들고 더하여 서버 수업시간에 aws(아마존)페이지에서 연 서버ip로 연동을하여 그 서버안에 있는 DB(mysql)에 
회원가입 정보,설문조사 결과를 저장하였습니다. 

---

### 개발 부분

- Mysql 연동, AWS서버 연동
- Gmail SMTP 메일 연동
